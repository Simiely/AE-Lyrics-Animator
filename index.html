01 / Animators

五个动画器，一键生成。

→
入场
文字从选定方向模糊进入。支持 4 方向、逐字/一起模式。
←
出场
文字反向渐隐消失。方向自动对称入场，逐字/一起可选。
〰
高度错落
逐字独立浮动，正弦波浪效果。幅度、频率、速度可调。
✦
散落分布
随机位置 + 大小 + 模糊。时间渐入可控，种子可复现。
↔
字间距动态
初始→结束字间距线性变化，逐字独立控制。基于 Tracking Amount 属性，与 AE 字符面板直觉一致。
02 / Params

灵活可控。

方向
4 向选择
左→右 / 右→左 / 上→下 / 下→上，出场自动对称
模式
逐字 / 一起
入场出场独立选择，逐字扫描或整段统一动画
开关
面板独立启用
五个动画器各有关闭开关，自由组合效果
字间距
初始→结束线性变化
逐字独立控制 Tracking Amount，缩放系数 0.1 与面板一致
模糊
随机模糊
基于种子概率决定模糊，可调节概率和范围
预设
双层持久化
工程目录 JSON + app.settings 全局保底，跟工程走随时可用
03 / Usage

三步上手。

STEP 01
📁
放入目录
将 .jsx 放入 AE ScriptUI Panels 目录
STEP 02
🪟
打开面板
AE 窗口菜单 → 歌词逐字散落动画工具
STEP 03
✨
点击应用
选中文本图层 → 调参 → 应用动画
