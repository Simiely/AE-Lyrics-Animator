# AE-Lyrics-Animator / 歌词逐字动画工具

AE 2026 歌词逐字动画脚本（ScriptUI Panel），当前版本 **v3.5**。

## 功能

选中文本图层，一键生成五种文字动画器，每个可独立开关：

| 动画器 | 效果 |
|--------|------|
| **入场** | 文字从选定方向模糊进入 → 清晰，支持逐字/一起模式 |
| **出场** | 文字反向渐隐消失 + 模糊，支持逐字/一起模式 |
| **高度错落** | 每个字符独立上下浮动，波浪效果 |
| **字间距动态** | 初始→结束字间距线性变化，逐字独立控制 |
| **散落分布** | 随机位置 + 随机大小 + 随机模糊，时间可控 |

## 安装

1. 将 `歌词逐字散落动画工具.jsx` 复制到 AE 的 ScriptUI Panels 目录：
   - **Windows**: `C:\Program Files\Adobe\Adobe After Effects 2026\Support Files\Scripts\ScriptUI Panels\`
   - **macOS**: `/Applications/Adobe After Effects 2026/Scripts/ScriptUI Panels/`
2. 重启 AE，通过 **窗口 > 歌词逐字散落动画工具 v3.5** 打开面板
3. 在 **编辑 → 首选项 → 脚本和表达式** 中开启 **允许脚本写入文件和访问网络**（预设持久化需要）

## 使用方法

1. 选中一个或多个文本图层
2. 在面板中调整参数，每个模块有独立启用开关
3. 点击 **应用动画**，自动在文本图层上生成对应 Animator
4. 再次应用会先清除旧的 `歌词_` 前缀动画器，再重新生成

### 预设管理

- **存储**：将当前参数保存到 4 个预设槽位之一
- **使用**：加载指定槽位的参数到面板
- **清除**：清空所有预设

预设采用**双层持久化**存储：

| 层级 | 存储位置 | 特点 |
|------|----------|------|
| 工程目录 JSON | `.aep` 文件同级目录 `歌词动画预设.json` | 跟工程走，不同工程独立 |
| 全局保底 | AE 偏好设置（`app.settings`） | 不依赖工程，随时可用 |

读取时优先读工程目录 JSON，不存在则回退到全局设置。

| **入场** | 持续时间 | 入场动画时长（秒） |

| | 偏移 | 入场方向偏移距离 |

| | 模式 | 逐字 / 一起 |
| **出场** | 开始时间 | 出场动画开始的绝对时间（秒） |
| | 持续时间 | 出场动画时长（秒） |
| | 偏移 | 出场方向偏移距离 |
| | 模式 | 逐字 / 一起 |
| **高度错落** | 幅度 | 上下浮动的像素值 |
| | 频率 | 波浪频率 |

| | 结束字间距 | 动画结束时的字间距值 |

| | 种子 | 随机种子（相同值可复现） |

| | 最小缩放 | 随机缩放下限（%） |

| | 最小模糊 | 模糊量下限 |
