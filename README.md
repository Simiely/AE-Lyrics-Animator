| 版本 | 文件 | 效果 |
|------|------|------|

## v1 — 歌词逐字动画工具

一个用于 **Adobe After Effects 2026** 的 ScriptUI 面板脚本，为歌词文本图层一键生成逐字动画效果。

### 功能

选中一个文本图层后，自动创建三种文字动画器：

| 动画器 | 效果 |
|--------|------|
| **入场** | 文字从左侧模糊进入 → 逐渐清晰，Opacity 从 0 → 100% 逐字扫描 |
| **出场** | 文字向右渐隐消失并变模糊，在指定时间开始 |
| **高度错落** | 每个字符独立上下浮动，形成波浪错落效果 |

## 附加特性

- **从播放头开始** — 动画从合成中当前时间指示器位置开始生成
- **预设管理** — 4 个存储槽位，可保存/加载参数配置
- **工程持久化** — 预设数据保存在 .aep 工程文件中，换电脑打开仍然保留
- **一键清除** — 清除所有已生成的动画器

## 使用方法

1. 将脚本文件放入 AE 的 ScriptUI Panels 目录：
   - Windows: `C:\Program Files\Adobe\Adobe After Effects 2026\Support Files\Scripts\ScriptUI Panels\`
   - macOS: `/Applications/Adobe After Effects 2026/Scripts/ScriptUI Panels/`
2. 在 AE 中通过 **窗口 > 歌词逐字动画工具** 打开面板
3. 在合成中选中一个文本图层，将播放头放到动画开始的位置
4. 调整参数，点击 **应用动画**
5. 点击 **清除动画** 移除已生成的动画器

> **提示：** 如文字显示不全，在文本末尾加空格即可。

## 参数说明

### 入场参数
- **持续时间 (秒)**：文字从模糊到清晰完全入场的时间
- **最大模糊值**：入场开始时文字的模糊程度
- **入场偏移 (像素)**：文字从左侧进入的偏移距离

### 出场参数
- **出场开始 (秒)**：出场动画开始的时间（从播放头位置算起的绝对时间）
- **出场持续时间 (秒)**：出场动画过渡时间
- **出场偏移 (像素)**：文字消失时向右的偏移距离

### 高度错落
- **波动幅度 (像素)**：字符上下浮动的最大距离
- **波动频率**：浮动速度
- **流动速度**：整体动画速率

## 预设管理

面板底部提供 4 组预设槽位：

```
存储 [1] [2] [3] [4] [清除全部]
使用 [1] [2] [3] [4] [复位]
```

- **存储行** — 点击数字保存当前面板参数到该槽位
- **使用行** — 点击数字加载已保存的参数（无数据时灰色不可点击）
- **复位** — 将所有参数恢复为默认值
- **清除全部** — 清空所有槽位

预设数据存储于工程 XMP 元数据中，随 .aep 文件跨设备迁移。

## v2.0 — 歌词逐字散落动画工具

完整的四合一动画工具：入场 + 出场 + 高度错落 + 散落分布。

### 功能

| 动画器 | 效果 |
|--------|------|
| **入场** | 文字从左侧模糊进入 → 逐渐清晰 |
| **出场** | 文字向右渐隐消失并变模糊 |
| **高度错落** | 每个字符独立上下浮动，形成波浪错落效果 |
| **散落分布** | 每个字符随机散布 + 随机大小 |

### 参数

#### 高度错落参数
- **波动幅度 (像素)** — 字符上下浮动的最大距离（默认 30）
- **波动频率** — 浮动速度（默认 0.7）
- **流动速度** — 整体动画速率（默认 1.0）

#### 散落分布参数
- **散布范围 (像素)** — 字符随机偏移的最大距离（默认 150）
- **随机种子** — 控制随机分布位置（默认 1），相同种子可复现
- **最小缩放 / 最大缩放 (%)** — 字符缩放的随机范围（默认 50%～200%）

### 使用方法

与 v1 相同，放入 AE ScriptUI Panels 目录即可。

> **提示：** 相同种子和参数下，每次生成的位置完全相同。改变种子可得到不同分布。
