# AE-Lyrics-Animator / 歌词逐字动画工具

AE 2026 歌词逐字动画脚本（ScriptUI Panel），当前版本 **v3.0**。

`歌词逐字散落动画工具.jsx` — 方向选择 + 入场 + 出场 + 高度错落 + 散落分布（时间可控）+ 随机模糊

---

## 功能

选中一个文本图层后，自动创建四种文字动画器：

| 动画器 | 效果 |
|--------|------|
| **入场** | 文字从选定方向（左/右/上/下）模糊进入 → 逐渐清晰 |
| **出场** | 文字向入场方向的反方向渐隐消失并变模糊 |
| **高度错落** | 每个字符独立上下浮动，形成波浪错落效果 |
| **散落分布** | 每个字符随机散布 + 随机大小 + 随机模糊，时间可控 |

## 附加特性

- **从播放头开始** — 动画从合成中当前时间指示器位置开始生成
- **预设管理** — 4 个存储槽位，可保存/加载参数配置
- **工程持久化** — 预设数据保存在 .aep 工程文件中，换电脑打开仍然保留
- **一键清除** — 清除所有已生成的动画器

## 使用方法

1. 将脚本文件放入 AE 的 ScriptUI Panels 目录：
   - Windows: `C:\Program Files\Adobe\Adobe After Effects 2026\Support Files\Scripts\ScriptUI Panels\`
   - macOS: `/Applications/Adobe After Effects 2026/Scripts/ScriptUI Panels/`
2. 在 AE 中通过 **窗口 > 歌词逐字散落动画工具 v3.0** 打开面板
3. 在合成中选中一个文本图层，将播放头放到动画开始的位置
4. 调整参数，点击 **应用动画**
5. 点击 **清除动画** 移除已生成的动画器

> **提示：** 如文字显示不全，在文本末尾加空格即可。

## 参数说明

### 入场参数
- **持续时间 (秒)** — 文字从模糊到清晰完全入场的时间（默认 2.0）
- **最大模糊值** — 入场开始时文字的模糊程度（默认 40）
- **入场偏移 (像素)** — 文字从选定方向进入的偏移距离（默认 80）
- **入场方向** — 下拉选择：从左到右 / 从右到左 / 从上到下 / 从下到上

### 出场参数
- **出场开始 (秒)** — 出场动画开始的绝对时间（默认 3.5）
- **出场持续时间 (秒)** — 出场动画过渡时间（默认 2.0）
- **出场偏移 (像素)** — 文字消失时的偏移距离（默认 80）
- 出场方向自动与入场方向对称

### 高度错落参数
- **波动幅度 (像素)** — 字符上下浮动的最大距离（默认 30）
- **波动频率** — 浮动速度（默认 0.7）
- **流动速度** — 整体动画速率（默认 1.0）

### 散落分布参数
- **散布范围 (像素)** — 字符随机偏移的最大距离（默认 150）
- **随机种子** — 控制随机分布位置（默认 1），相同种子可复现
- **散落开始 (秒)** — 散落效果开始时间，应 ≥ 入场持续时间避免重叠（默认 2.0）
- **散落过渡 (秒)** — 散落从无到完全散开的过渡时间（默认 1.0）
- **最小缩放 / 最大缩放 (%)** — 字符缩放的随机范围（默认 50%～200%）

### 随机模糊参数
- **模糊随机种子** — 控制模糊分布的随机种子（默认 10）
- **模糊概率 (%)** — 多少比例的字符会被模糊（默认 40）
- **最小模糊值 / 最大模糊值** — 模糊程度的随机范围（默认 0～25）

> **提示：** 模糊概率设为 0 可完全关闭随机模糊。散落开始时间 + 散落过渡时间控制散落效果的时序，配合入场持续时间可精确编排。

## 预设管理

面板底部提供 4 组预设槽位：

```
存储 [1] [2] [3] [4] [清除全部]
使用 [1] [2] [3] [4] [复位]
```

- **存储行** — 点击数字保存当前面板参数到该槽位
- **使用行** — 点击数字加载已保存的参数（无数据时灰色不可点击）
- **复位** — 将所有参数恢复为默认值
- **清除全部** — 清空所有槽位

预设数据存储于工程 XMP 元数据中，随 .aep 文件跨设备迁移。
