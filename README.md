# AE-Lyrics-Animator / 歌词逐字动画工具

AE 2026 歌词逐字动画脚本（ScriptUI Panel），当前版本 **v3.4**。

## 功能

选中文本图层，一键生成四种文字动画器，每个可独立开关：

| 动画器 | 效果 |
|--------|------|
| **入场** | 文字从选定方向模糊进入 → 清晰，支持逐字/一起模式 |
| **出场** | 文字反向渐隐消失 + 模糊，支持逐字/一起模式 |
| **高度错落** | 每个字符独立上下浮动，波浪效果 |
| **散落分布** | 随机位置 + 随机大小 + 随机模糊，时间可控 |

## 使用方法

1. 放入 AE ScriptUI Panels 目录，通过 **窗口 > 歌词逐字散落动画工具 v3.4** 打开
2. 选中文本图层，调好参数，点击 **应用动画**

## 参数速览

| 分类 | 参数 |
|------|------|
| 入场 | 持续时间、最大模糊、偏移、方向(4向)、模式(逐字/一起) |
| 出场 | 开始时间、持续时间、偏移、模式(逐字/一起) |
| 高度错落 | 幅度、频率、速度 |
| 散落分布 | 范围、种子、开始时间、过渡时间、缩放范围、模糊参数 |

每个面板有独立启用复选框，预设可存储 4 组随工程持久化。
